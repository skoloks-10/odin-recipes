# Odin Recipes

## 프로젝트 소개
The Odin Project의 HTML 기초 프로젝트입니다. 기본적인 HTML 구조와 태그를 사용하여 간단한 레시피 웹사이트를 만드는 프로젝트입니다.

## 프로젝트 목표
이 프로젝트를 통해 다음을 학습합니다:
- HTML 기본 구조 (boilerplate) 작성
- 제목 태그 (`<h1>`, `<h2>` 등) 사용
- 링크 태그 (`<a>`) 사용 및 상대 경로 이해
- 이미지 태그 (`<img>`) 사용
- 순서 있는 목록 (`<ol>`)과 순서 없는 목록 (`<ul>`) 사용
- 기본적인 웹사이트 구조와 네비게이션 구현

## 프로젝트 구조
```
odin-recipes/
├── index.html
├── README.md
└── recipes/
    ├── recipe1.html
    ├── recipe2.html
    └── recipe3.html
```

## 구현 기능
- **메인 페이지 (index.html)**: 모든 레시피로 연결되는 링크 목록
- **레시피 페이지**: 각 레시피의 상세 정보
    - 완성된 요리 이미지
    - 요리 설명
    - 재료 목록 (순서 없는 목록)
    - 조리 단계 (순서 있는 목록)
    - 메인 페이지로 돌아가는 링크

## 개발 과정

### 1차 반복: 초기 구조
- `index.html` 파일 생성
- HTML boilerplate 코드 작성
- "오딘 레시피" 제목 추가

### 2차 반복: 레시피 페이지
- `recipes` 디렉토리 생성
- 첫 번째 레시피 HTML 파일 생성
- 메인 페이지와 레시피 페이지 간 링크 연결

### 3차 반복: 레시피 콘텐츠
- 레시피 이미지 추가
- 요리 설명 작성
- 재료 목록 추가 (순서 없는 목록)
- 조리 단계 추가 (순서 있는 목록)

### 4차 반복: 레시피 확장
- 추가 레시피 2개 생성 (총 3개)
- 메인 페이지의 링크를 순서 없는 목록으로 구조화

## 사용 기술
- HTML5

## 배운 점
- HTML의 기본 구조와 시맨틱 태그의 중요성
- 상대 경로를 사용한 파일 간 링크 연결 방법
- 목록 태그를 활용한 콘텐츠 구조화
- 기본적인 웹사이트 네비게이션 구현

## 향후 개선 사항
- CSS를 활용한 스타일링
- 반응형 디자인 적용
- JavaScript를 활용한 인터랙티브 기능 추가
